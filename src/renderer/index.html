📖 使用说明

坚果云配置:

  1. 登录坚果云网页版
  2. 进入 账户信息 → 安全选项 → 第三方应用管理
  3. 添加应用密码，名称填 "AnotherSSH"
  4. 服务器地址: https://dav.jianguoyun.com/dav/
  5. 用户名: 你的坚果云账号邮箱
  6. 密码: 刚才生成的应用密码

⚠️ 首次同步:

由于坚果云限制，首次使用需要先在坚果云中手动创建配置文件：

  1. 打开坚果云网页版或客户端
  2. 在根目录创建一个空的文本文件
  3. 文件名与上面"远程文件路径"保持一致
  4. 然后点击"立即同步"即可
