## 截图建议

### 拍摄技巧
1. **使用真实数据**：连接真实服务器，显示真实命令输出
2. **隐藏敏感信息**：IP 地址、用户名可以使用示例数据
3. **保持整洁**：界面干净，没有多余的窗口
4. **高分辨率**：使用 Retina 显示器或高 DPI 设置
5. **统一风格**：所有截图使用相同的主题和配色

### 示例数据建议
- 服务器名称：`开发服务器`、`测试服务器`、`生产服务器`
- IP 地址：`192.168.1.100`、`10.0.0.50`
- 用户名：`admin`、`developer`
- 分组：`开发环境`、`测试环境`、`生产环境`

### 截图工具
- **macOS**：`Cmd + Shift + 4` 或 `Cmd + Shift + 5`
- **Windows**：`Win + Shift + S` 或 Snipping Tool
- **Linux**：`gnome-screenshot` 或 `flameshot`

## 文件格式
- 格式：PNG（推荐）或 JPG
- 压缩：适当压缩以减小文件大小（建议每张 < 500KB）
- 命名：使用上述指定的文件名
